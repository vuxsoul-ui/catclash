# CatClash Soul

## Core Identity
CatClash is a mobile-first, AFK PvP cat arena game built around:
- Visual-first voting
- Fair match rotation
- Low-latency swipe interactions
- Rarity-based progression

## Non-Negotiables
- Mobile-first UX
- Swipe physics must feel premium
- Fair match distribution
- Low Supabase egress footprint
- Progressive disclosure UI (clean front, rich back)

## Philosophy
- Instant gratification
- Low cognitive load
- Casual but competitive
- Animations > clutter

## Technical Constraints
- Next.js App Router
- Supabase backend
- Public storage bucket with derivatives
- Low egress strategy
