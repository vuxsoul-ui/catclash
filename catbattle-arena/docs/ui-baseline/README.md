## Top 5 Mobile Pain Points
1. Toast and transient notices are bottom-anchored near the nav and can compete with tap targets during active voting.
2. Header + bottom nav consume too much vertical space on iPhone, reducing above-the-fold gameplay density.
3. Above-the-fold on Home is visually busy (many blocks/chips), slowing speed-to-first-vote.
4. Duel and Shop cards have inconsistent spacing/button sizing, creating a less premium feel and scan friction.
5. Motion/effects are inconsistent by context (some interactions feel static while others are intense), reducing cohesive “event-like” feedback.
